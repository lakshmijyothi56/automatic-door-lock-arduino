# Automatic Door Lock System using Arduino

## Project Description
This project uses an Arduino Uno, servo motor, push button, and LED to create an automatic door lock system.

## Components Used
- Arduino Uno
- Servo Motor
- Push Button
- LED
- 220 Ohm Resistor
- Breadboard
- Jumper Wires

## Working Principle
When the push button is pressed, the Arduino sends a signal to the servo motor to rotate 90 degrees and open the door. The LED glows to indicate the door is open. After 3 seconds, the servo returns to 0 degrees and closes the door.
